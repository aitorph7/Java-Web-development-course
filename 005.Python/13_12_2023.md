Crear una lista:

students = []

Acceder a elementos de una lista:

students[0]
students[-1]

Modificar elementos:

students[0] = "nuevodato"

Agregar elementos a una lista:

* append()
* insert()

Borrar elementos de una lista:

* remove()
* pop()
* del student[2]

Ordenar listas:

* .sort()
* .sort(reverse=True)
* sorted(students) --> Devuelve una nueva lista ordenada sin modificar la original

Invertir elementos:

* .reverse()
